# Tunnel Runner 🎮

Ein 2D-Tunnel-Runner inspiriert vom General Grievous Wheel Bike Minispiel aus LEGO Star Wars: The Complete Saga (Nintendo DS).

## Gameplay
- Das Fahrzeug fährt auf der **Innenwand** eines 360°-Tunnels
- **Steuerrad** dreht die Position im Tunnel
- **Sprung-Button / Leertaste** = Springen
- 🟤 Braune Blöcke → **Ausweichen**
- 🔵 Cyan-Ringe → **Springen**
- 🪙 Goldene Münzen → Einsammeln für Punkte

## Stack
- HTML5 Canvas + JavaScript
- Einzelne `index.html` (keine Build-Tools nötig)
- Keine externen Libraries außer Google Fonts

## Entwicklung
Einfach `index.html` im Browser öffnen (oder mit Live Server in VSCode).
